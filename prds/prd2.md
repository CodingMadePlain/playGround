# Application Creation

- Create a simple example of a small web application with the following goal:

- A user submits feedback through a form. The feedback is stored in a JSON file on a hosted server. A dashboard running on a local server then reads that JSON file and displays the submitted feedback.

- Use simple technologies suitable for beginners:

* PHP on the hosted server to save and read the JSON file
* HTML (and PHP if needed) on the hosted server to receive the form data
* HTML, CSS and vanilla JavaScript on the local server to display a dashboard
* JavaScript should use `fetch()` to request the JSON data from the hosted server

- Explain the structure of the solution clearly and simply. Include:

• A short description of how the system works
• The files needed on the hosted server
• The files needed on the local server
• Example code snippets showing how the form sends data and how JavaScript reads the JSON file
• A simple example of what the JSON data might look like

- Keep the explanation beginner-friendly and avoid unnecessary complexity. The purpose is to help students start building the application themselves and understand how the pieces connect.
