# Create the following skill

- Use the following data to create me a agent skill. Follow the documentation at [Use Agent Skills in VS Code](https://code.visualstudio.com/docs/copilot/customization/agent-skills)

## Objective
This skill is to help in the building of web pages from presented designs or frameworks or sketches.

- Where exact images are not avialable for the design use placeholders from [placehold.co](https://placehold.co/)

- for example

```html
https://placehold.co/600x400/000000/FFFFFF/png
```

- Where exact dimensions are not providedfor the layout, please ask the user to provide before proceeding

- Always check that a images or framework has been provied before proceding
- All html pages must be responsive
