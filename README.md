# Project Overview

**Trainer:** Dele Oke

> Cover [Introduction notes](notes/introduction.md)

This project demonstrates how to use AI Assistants to build web applications inside VS Code.

## Purpose

The application fetches remote content (CSV and JSON files) from external servers and saves it to the local machine. JavaScript then reads those files and displays the data in an HTML page served via the VS Code Live Server extension.

The project also covers generating website templates from images and wireframes, and processing images using the Python Pillow package.

## Technology Stack

- **Server:** LAMP (Linux, Apache, MySQL, PHP)
- **Backend:** PHP 8.3, Python 3.11+
- **Database:** MySQL 8.0+
- **Frontend:** HTML5, vanilla CSS3, vanilla JavaScript (ES6+)
- **Version control:** Git 2.23+
